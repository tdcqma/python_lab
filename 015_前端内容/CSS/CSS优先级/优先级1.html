1. 测试需求：让"优先级"这三个字变红
  2. CSS样式北浏览器解析优先级顺序：
    • 有[!important]的时候，不计算权值，优先级高于一切。
    • 文内样式，内联样式表中的权值最高： style="" ,权值：1000
    • 统计选择符ID：id="" ,权值：100
    • 统计选择符CLASS：class="" ,权值：10
    • 统计选择符HTML标签：p="" ,权值：1
    • 优先级一样的时候，style里描述的后边的样式覆盖前边的样式
优先级测试

测试2:
需要进行优先级权值的计算后，才知道"嵌套优先级"到底显示什么颜色。

经过计算，嵌套优先级 显示粉色
>>div1 .div3 是110的权值
>>div1 .div2 div 是111的权值，
所以，最后选择粉色
测试3：
有时候优先级加来加去，自己会不清楚到底是哪里起的css样式效果，就想指定显示某一颜色而忽略优先级的时候，添加【!important】后即可实现。
  • 理论上，[.div4 .div5的优先级权值是20（红色），单独一个.div5是10（蓝色），通过权值来看显示红色]
  • 但是，在[.div5后面加上"！important"]后，虽然权值低，也强制显示蓝色了。
